SkillCourse — Образовательная платформа

Тема проекта:
Образовательная платформа / курсы.
Описание проекта

SkillCourse — это современный многостраничный веб-сайт, разработанный для образовательной платформы, специализирующейся на IT-обучении. Проект объединяет в себе удобный интерфейс для выбора учебных программ, актуальные новости индустрии и информацию о карьерном старте для новичков. Сайт ориентирован на предоставление качественного пользовательского опыта через чистый дизайн и понятную навигацию.
Список страниц сайта

Сайт состоит из 5 взаимосвязанных страниц:

    Главная страница (index.html) — обзор популярных курсов, языков программирования и последних новостей блога.

    Каталог курсов (pages/catalog.html) — полный перечень образовательных программ.

    О платформе (pages/about.html) — информация о миссии проекта и экспертах.

    Контакты (pages/contacts.html) — контактные данные, адрес офиса и форма обратной связи.

    Регистрация/Вход (pages/register.html) — интерфейс для авторизации новых пользователей.

Используемые технологии

    HTML5: Использование семантической разметки (header, nav, main, section, article, footer) для обеспечения доступности и SEO.

    CSS3: Применение современной архитектуры разделения стилей и адаптивной верстки.

        Flexbox & CSS Grid: Для построения сложных сеток и выравнивания элементов.

        Многофайловая архитектура: Разделение на reset.css, base.css, layout.css и components.css.
